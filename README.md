# Emmanuel Mukumbwa — Personal Portfolio (React)

This repository is a React-based personal portfolio and projects website for Emmanuel aRelic Mukumbwa. It showcases projects, testimonials, a resume page, and contact functionality. The site is built with Create React App and structured for easy development and deployment (Vercel-friendly).

## What this project does

- Presents a homepage with featured projects and call-to-action sections.
- Lists project detail pages (images, descriptions, links).
- Shows testimonials and a downloadable resume.
- Provides a contact form to reach out.
- Includes theme support (light/dark) and responsive UI components.

## Tech stack

- React (Create React App)
- JavaScript (ES6+)
- CSS / SCSS for styling
- Vercel-ready static build (output in `build/`)

## Project structure (key files)

- `src/` — source files (components, pages, assets)
- `public/` — public static files
- `build/` — production-ready build (generated by `npm run build`)
- `package.json` — scripts & dependencies

## Quick local development

1. Install dependencies:

```powershell
npm install
```

2. Start the dev server:

```powershell
npm start
```

Open http://localhost:3000 to view the site. The dev server supports hot reload.

## Build for production

```powershell
npm run build
```

The command produces an optimized static bundle in the `build/` folder which can be deployed to Vercel, Netlify, GitHub Pages, or any static host.

## Deployment notes

- This repo includes a `vercel.json` file and is ready to deploy to Vercel with default settings.
- When deploying, ensure environment variables (if any) used by contact form or analytics are set in the hosting platform.
